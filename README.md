# AutoTaxes by Dawlat

**Версия**: 1.0.0  
**Автор**: Dawlat  
**Платформа**: [MoonLoader](https://blast.hk/moonloader/)

AutoTaxes — это скрипт на LUA для автоматизации процесса оплаты налогов на серверах [Arizona RP](https://arizona-rp.com/). С помощью скрипта можно автоматически оплачивать выбранную группу налогов при входе в игру или активировать автоматическую оплату используя предустановленные команды.

## Особенности

- Авто-оплата обычных налогов при входе в игру
- Авто-оплата налогов за семейную квартиру при входе в игру
- Авто-продление аренды отеля при входе в игру
- Настройка задержки выполнения после авторизации
- Настройка задержки между действиями
- Возможность оплатить налоги по команде в чат

## Установка

1. Скачайте файл `AutoTaxes v1.0.0 by Dawlat` из [репозитория](https://github.com/OblivionGM/AutoTaxes-by-Dawlat).
2. Переместите файл `AutoTaxes v1.0.0 by Dawlat` в папку `moonloader` вашей SAMP-сборки.
3. Перезапустите игру если она была запущена.

## Использование

1. Для открытия настроек используйте команду `/tax`
2. Для оплаты всех налогов - `/pall`
3. Для оплаты обычных налогов - `/ptax`
4. Для оплаты семейных налогов - `/pftax`
5. Для продления номера в отеле - `/phtax`

Рекомендации:
- Оптимальная задержка между действиями - 250мс. Изменяйте её в большую сторону при задержках сервера и/или нестабильном интернет соединении
- При настройке автоматической оплаты нескольких видов налогов оставляйте задержку между оплатой минимум 3 секунды

Примечание: При автоматической оплате налогов контроллируйте выполнение оплаты по сообщениям в чате.
В следствии ошибок, обновлений игры и прочих факторов оплата может не сработать.
